# ResQNet

Realtime disaster response and emergency coordination platform.

## Features

- SOS emergency reporting
- Realtime rescue coordination
- Flood simulation using BFS
- Route optimization using Dijkstra
- Live disaster map
- Shelter management
- Role-based dashboards

## Tech Stack

- React + Vite
- Node.js + Express
- MongoDB
- Socket.IO
- Leaflet Maps
- Tailwind CSS
